> geno-research

Research skills for Claude Code. Deep research workflows, lab notes, paper generation, documentation, and autonomous research loops.

Commands

CommandDescription
/gt-research <brief> Deep research with multi-resolution Obsidian knowledge graphs (L0-L3 notes, literature review, canvas)
/gt-lab-notes <subcommand> Project lab notes: create, add-task, do-task, done-task, note
/gt-gen-paper [focus] Generate academic paper from benchmark findings
/gt-repo-docs [focus] Generate purpose-driven repo documentation
/gt-supercharge [duration] Long-running autonomous agent loop with structured cycles

Install

# Direct install
./install.sh # Or via geno-tools ecosystem installer
cd ../geno-tools && ./install.sh

Ecosystem

geno-tools -- orchestrator + general tools geno-kaggle -- Kaggle benchmarking geno-media -- media creation (audiobooks, video) geno-research -- research workflows (this repo)
